this is a batch file that is a muultipurpuse IT helping hand.
when u start the bat file it wil look something like this:
==== SYSTEM TOOLS (A) ====
A1 - Show IP Configuration
A2 - Show System Information
A3 - Task Manager
A4 - Command Prompt
A5 - Open Calculator
A6 - Open Notepad
A7 - System File Checker (sfc /scannow)
A8 - Disk Check (chkdsk)
A9 - Device Manager
A10- Task Scheduler

==== NETWORK TOOLS (B) ====
B1 - Show WiFi Profiles
B2 - Show WiFi Password
B3 - Ping Google
B4 - Show ARP Table
B5 - Show Network Connections
B6 - Clear DNS Cache
B7 - Show Network Adapters

==== USER TOOLS (C) ====
C1 - Show Current User
C2 - Rename User
C3 - List Local Users
C4 - Change Computer Name
C5 - List Installed Programs

==== UTILITIES / OTHER (D) ====
D1 - Open Control Panel
D2 - Open File Explorer
D3 - Change Console Colors
D4 - Battery Report
D5 - List Running Processes
D6 - Show Running Services
D7 - Clean Temporary Files

==== POWER (E) ====
E1 - Shutdown
E2 - Restart
E3 - Log Off
E4 - Exit
